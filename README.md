## Installation

1. Clone the repo

   ```
   git clone https://github.com/cadellsingh/movieAwards.git
   ```

2. Change working directory

   ```
   cd movieAwards/
   ```

3. Install any dependencies

   ```
   yarn install
   ```

4. Run application

   ```
   yarn start
   ```
